## 🚀 QUICK START (5 minutes)

### **Step 1: Setup (Automated)**

**On Windows:**
```bash
setup.bat
```

**On Mac/Linux:**
```bash
chmod +x setup.sh
./setup.sh
```

⏳ This installs all dependencies automatically (~3-5 minutes)

### **Step 2: Start Backend**

Open **Terminal 1** and run:
```bash
cd backend
npm run dev
```

✅ You'll see: `✓ Server is running on port 5000`

### **Step 3: Start Frontend**

Open **Terminal 2** and run:
```bash
cd frontend
npm run dev
```

✅ You'll see: `Local: http://localhost:3000`

### **Step 4: Open Browser**

Visit: **http://localhost:3000**

### **Step 5: Test the App**

1. **Register** - Create a new account
2. **Add Pet** - Click "My Pets" tab, add your pet
3. **Book Appointment** - Click "Appointments" tab, schedule a vet visit

## 🆘 Troubleshooting

**Backend won't start:**
```bash
cd backend
npm install
npm run dev
```

**Frontend won't start:**
```bash
cd frontend
npm install
npm run dev
```

**Port already in use:**
- Change ports in .env (backend)
- Vite auto-selects frontend port

**Database errors:**
```bash
# Delete database and restart
rm backend/petcare.db
npm run dev
```

See [DEVELOPER_GUIDE.md](DEVELOPER_GUIDE.md) for more troubleshooting.
